## 作為擴充功能使用

可以在 MakeCode 中將此儲存庫新增為**擴充功能**。

* 開啟 [https://makecode.microbit.org/](https://makecode.microbit.org/)
* 按一下**新專案**
* 按一下工具齒輪選單下的**擴充功能**
* 搜索 **https://github.com/ylcdolphin/easymaqueen** 並匯出

## 編輯此專案

* 開啟 [https://makecode.microbit.org/](https://makecode.microbit.org/)
* 按一下**匯入**，然後按一下**匯入 URL**
* 貼上 **https://github.com/ylcdolphin/easymaqueen** 並按一下匯入
